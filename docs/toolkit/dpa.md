## Appendix 13.22 â€” Sample Data Processing Clauses (Non-legal; to be reviewed)

Insert into simple contractor or supplier contracts where data is processed (e.g., box office / ticketing)
- Data Processing: Supplier processes personal data as a processor on behalf of the Project.
- Purpose: Supplier will process data only for ticketing and customer-service purposes.
- Security: Supplier to keep data encrypted in transit and at rest, maintain MFA for admin accounts, and use secure backups.
- Sub-processors: Supplier to notify Project of new sub-processors with an opt-out clause.
- Deletion: Supplier will return or delete personal data upon contract termination, except for financial records kept for 7 years for audit.
- Breach notification: Supplier will notify the Project within 24 hours of confirming a reportable actual breach.
