    How to Use This Index

    Finding Visual Elements: - Click any item title to jump directly to that diagram, table, or template - Section references (e.g., #appendix-13-9) link to the containing section - Mermaid diagrams render interactively in supported viewers - ASCII diagrams display universally in monospace font

    Printing & Export: - Tables are print-optimised with clear borders and headers - Diagrams include detailed text descriptions for accessibility - Templates can be extracted and used standalone

    Customisation: - All templates are editable examples; adapt to your context - Mermaid diagram code can be modified and re-rendered - ASCII diagrams can be updated in any text editor
