    Useful Contacts and Resources

    Regulatory and Statutory Bodies

    Southwark Council - Main: southwark.gov.uk | 020 7525 5000 - Planning: planning.southwark.gov.uk - Licensing: southwark.gov.uk/licensing - Building Control: building.control@southwark.gov.uk - Culture Team: culture@southwark.gov.uk

    Transport for London (TfL) - Property enquiries: property@tfl.gov.uk - General: tfl.gov.uk | 0343 222 1234

    Network Rail - Asset Protection: networkrail.co.uk/running-the-railway/looking-after-the-railway/asset-protection-and-optimisation/ - General: networkrail.co.uk | 03457 11 41 41

    Historic England - Website: historicengland.org.uk - Advice line: 0370 333 0607 - Listed building and conservation area guidance

    Using This Contact Matrix:

    1. Add to Project Documentation: Copy this table into your project management system (Asana, Trello, Monday.com) or create a shared spreadsheet for team access
    2. Track Engagement: Add columns for "Last Contact Date," "Status," "Action Required" to monitor relationship development
    3. Personalise: Replace generic email addresses with named contacts as relationships develop
    4. Update Regularly: Review quarterly—contacts change, new organisations emerge, priorities shift
    5. Integration: Link contact records to decision log (Appendix 13.15), RACI matrix (Appendix 13.15), and risk register

    Toolkit & Templates (Appendix 13.25)

    The Manual includes a set of practical templates and checklists that you can copy, adapt or print as required. These are intended to be reused during Discovery and Implementation. A small static site is included in the project and can be built with Docker (see README_SITE.md). - Appendix 13.8: Funder report template - Appendix 13.9: Health & Safety checklists (performance days) - Appendix 13.10: Closure & Handback checklist - Appendix 13.11: Rail Safety Addendum - Appendix 13.12: GDPR & DPA guidance - Appendix 13.13: Accessibility & Inclusion checklist - Appendix 13.14: Director's Core Playbooks (7-Day & 30/90 Day) - Appendix 13.15: RACI & Decision Log - Appendix 13.16: Statutory milestone & procurement triggers - Appendix 13.18: Budget Sensitivity tables - Appendix 13.20: Procurement & Tender evaluation checklist - Appendix 13.21: Sample MoU - Appendix 13.22: Sample DPA clauses

    • Printable & downloads
    • If using this document as a static PDF: extract the relevant appendix pages and print as a single-sheet checklist. -- For repositories: create a toolkit/ folder with the individual templates (MoU, DPA, Partner Contact Form, Daily H&S, Tender scoring matrix) and link them into each Appendix. When using a repo, mark each file as -sample to avoid accidental reuse as a final legal form.
    • Toolkit location (example): /Users/craig/Downloads/toolkit (move into repo root before publishing).
    • Note: When moving the toolkit into a Git repository, avoid committing personal data or signed MoUs containing personal info. Store sensitive documents in a secure location and add them to .gitignore if needed.
